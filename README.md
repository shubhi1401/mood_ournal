# 🌸 Mood Journal

**Track your emotional journey, one emoji at a time.**

Mood Journal is a beautifully designed Android application that helps users monitor their mental well-being through mood tracking, journaling, insightful analytics, and music therapy. By combining simple daily logging with data visualization and personalized recommendations, Mood Journal encourages emotional awareness and self-reflection.

---

## 📱 Features

### 📖 Daily Journaling

* **Quick Log:** Record your mood instantly with a single tap from the home screen.
* **Detailed Entries:** Add personal notes to capture thoughts, experiences, and emotions.
* **History Timeline:** Browse all previous mood entries in a clean chronological view with easy deletion support.

### 📊 Smart Insights

* **Mood Distribution:** Visualize emotional patterns through dynamic pie charts.
* **Personalized Quotes:** Receive motivational or comforting quotes based on your most frequently logged mood.
* **Streak Tracking:** Stay consistent with a daily mood logging streak counter.

### 📅 Interactive Mood Calendar

* **Mood Mapping:** View moods directly on a color-coded calendar.
* **One-Tap Review:** Tap any date to instantly view mood details, notes, and timestamps.

### 🎵 Music Therapy

* **Mood-Based Playlists:** Get curated song recommendations based on your current mood.
* **YouTube Integration:** Open songs directly on YouTube for an immersive listening experience.

### 🌓 Personalized Experience

* **Theme Toggle:** Switch between Light Mode and Dark Mode using the custom 🌸 theme button.
* **Modern UI:** Built with Material Design principles, CardViews, and smooth animations for an intuitive user experience.

## 🚀 How to Run the Project

### 1. Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/Mood-Journal.git
```

### 2. Open in Android Studio

* Launch Android Studio.
* Select **File → Open**.
* Navigate to the cloned project folder.

### 3. Sync Gradle

* Allow Android Studio to download dependencies and complete Gradle synchronization.

### 4. Run the Application

* Connect a physical Android device or start an emulator.
* Click the **Run ▶** button.
